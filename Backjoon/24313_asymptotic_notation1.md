## 알고리즘 수업 - 점근적 표기 1
### 문제
알고리즘의 소요 시간을 나타내는 O-표기법(빅-오)을 다음과 같이 정의하자.

```
O(g(n)) = {f(n) | 모든 n ≥ n0에 대하여 f(n) ≤ c × g(n)인 양의 상수 c와 n0가 존재한다}
```
이 정의는 실제 O-표기법과 다를 수 있다.
함수 f(n) = a1n + a0, 양의정수 c, n0가 주어질 경우 O(n)의 정의를 만족하는지 알아보자.

### 입출력
- 입력 :
  첫째 줄에 함수 f(n)을 나타내는 정수 a1, a0가 주어진다. (0 ≤ |ai| ≤ 100)
  다음 줄에 양의 정수 c가 주어진다. (1 ≤ c ≤ 100)
  다음 줄에 양의 정수 n0가 주어진다. (1 ≤ n0 ≤ 100)
- 출력 :
  f(n), c, n0가 O(n) 정의를 만족하면 1, 아니면 0을 출력한다.


### 풀이과정
#### 문제 해석
주어진 식 
1. f(n) = a1n + a0
2. O(n) -> g(n) = n

-> g(n) = n 대입
-> 모든 n ≥ n0에 대하여 f(n) ≤ c × n를 만족해야한다.

결국 주어진 f(n)에 대해 n ≥ n0 모든 구간에서 f(n) ≤ c × n을 만족하는지 확인해야한다.

#### 결과 도출
a0​ ≤ (c−a1​)n
여기서 n이 커질 수록 어떻게 변하는가? 
c-a1이 양수거나 0 일경우는 n이 증가할때 우변이 증가하므로 n0에서만 확인하면 된다.
음수일경우에는 n이 증가할수록 우변이 감소하므로 어떤 n ≥ n0에서는 부등식을 만족하지 못한다.

if문 조건
좌측 f(n)의 기울기인 a1이 우변 기울기 c보다 가파르게 증가하지 않는지를 체크 (a1<=c) 가파르지 않아야 성립.
&&
시작점 n0에서 실제로 만족하는가?

#### 최종 코드
```
import java.util.Scanner;

public class Main{
    public static void main(String args[]){
        Scanner sc = new Scanner(System.in);

        int a1 = sc.nextInt();
        int a0 = sc.nextInt();
        int c = sc.nextInt();
        int n0 = sc.nextInt();

        if(a1 <= c && a1 * n0 + a0 <=c * n0){
          System.out.println(1);
        } else {
           System.out.println(0);
        }
    }
}
```

### O(n) 판별 패턴
1. 비교 대상을 확정하기
   - O(n) -> g(n) = n
2. 정의에 대입하기
3. 식 정리하기
4. 기울기 체크
5. n0에서 확인 후 최종 판단
